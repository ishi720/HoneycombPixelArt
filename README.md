## セットアップ

```bash
# リポジトリをクローン
git clone https://github.com/ishi720/HoneycombPixelArt.git

# ディレクトリに移動
cd HoneycombPixelArt

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev
```

ブラウザで http://localhost:5173 を確認
